# Infrastructure Modules

This folder contains Bicep modules for provisioning Azure resources for the Zava Storefront application.

- `main.bicep`: Root orchestration template
- `modules/acr.bicep`: Azure Container Registry
- `modules/logAnalytics.bicep`: Log Analytics Workspace
- `modules/appService.bicep`: App Service Plan + Web App

See the root README and `azure.yaml` for deployment instructions.
